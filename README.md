# Google-Drive-Security-Audit-
Аудит прав доступа к Google Диску

1. Описание задачи (Problem Case)
В современных компаниях сотрудники часто открывают доступ к документам по ссылке или конкретным лицам. Со временем накапливается сотни файлов с внешним доступом, что создает угрозу информационной безопасности. Данный скрипт автоматизирует процесс аудита, формируя отчет обо всех файлах с доступом «для всех» или «внешним пользователям».

3. Техническое решение (Сценарий)
Используется Google Apps Script для сканирования Google Диска и Google Sheets для вывода отчета.

Основные функции:
Сканирование всех файлов владельца.
Определение типа доступа (Anyone, Restricted, Domain).
Проверка списка редакторов и читателей.
Вывод ссылок на подозрительные файлы в таблицу.

4. Инструкция по установке
  4.1 Создайте новую Google Таблицу.
  4.2 Перейдите в меню Расширения -> Apps Script.
  4.3 Скопируйте и вставьте следующий код, он приведен в файле audit_script.js
  4.4 Нажмите Сохранить (иконка дискеты).
  4.5 Обновите страницу с таблицей — появится пункт меню Безопасность.
  4.6 Нажмите «Запустить аудит прав» и предоставьте разрешения скрипту.
   
5. План действий при обнаружении угроз (Кризисный план)
   
   В репозитории также приложена схема действий:
Классификация: Определить критичность данных в файле.
Изоляция: Изменить настройки доступа на "Ограничено" (Restricted).
Уведомление: Сообщить ИТ-отделу, если файл содержал пароли или ПДн.
